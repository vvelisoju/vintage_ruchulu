1. Overview

Vintage Ruchulu is a food brand based in Warangal, offering authentic, homemade Non-Veg Pickles and (in future) other vintage food items.

This project is a fully static React web app that allows customers to browse products, add them to cart, and place an order via WhatsApp. No backend or database will be used; all data (products, images, prices, categories) is stored within the React codebase.

The app is optimized for mobile-first usage, simple UI, easy ordering, and fast static deployment (e.g., Replit, Netlify, Vercel).

2. Goals
Primary Goals

Provide a static, fast-loading React app to showcase the product catalog.

Enable customers to build a cart and place an order via WhatsApp.

Make editing prices, images, and product lists easy from within the code.

Include brand identity and trust-building content for Vintage Ruchulu.

Secondary Goals

Prepare scalable structure to add future categories (e.g., sweets, snacks).

Support repeat orders with prefilled customer info.

Maintain a simple, affordable ordering process with no backend cost.

3. Target Users
